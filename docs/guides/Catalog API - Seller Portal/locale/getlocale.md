## Response body has the following properties:

| Attribute | Type   | Description   |
| --------- | ------ | ------------- |
| data      | string | Store locale. |

## Response body example:
[block:code]
{
  "codes": [
    {
      "code": "\"pt-BR\"",
      "language": "json"
    }
  ]
}
