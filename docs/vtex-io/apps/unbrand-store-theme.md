Passo 1 - SCSS
Existe uma estrutura de SCSS dentro do projeto que será utilizada, NÃO EDITE DIRETAMENTE O CSS.

Para rodar o SCSS execute yarn sass ou npm run sass que ele ficará assistindo todas as mudanças.
