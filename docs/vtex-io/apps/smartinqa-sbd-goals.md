### Login in the vtex account

```bash
    vtex login [account-name]
    ex. vtex login smartinqa
```

### Create a workspace

```bash
    vtex use [workspace-name]
    ex. vtex use myworkspace
```

### Link the workspace

```bash
    vtex link
```
