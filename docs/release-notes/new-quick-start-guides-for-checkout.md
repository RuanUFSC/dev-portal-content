We have created quick-start guides to provide more details on how the Checkout APIs work and use them. Initially, we are making available the following guides from the [Shopping cart section](https://developers.vtex.com/docs/guides/checkout-api-overview#shopping-cart):

- [Simulate a shopping cart](https://developers.vtex.com/vtex-rest-api/docs/simulate-a-shopping-cart)
- [Create a new cart](https://developers.vtex.com/vtex-rest-api/docs/create-a-new-cart)
- [Get cart information by ID](https://developers.vtex.com/vtex-rest-api/docs/get-cart-information-by-id)
- [Remove all items from cart](https://developers.vtex.com/vtex-rest-api/docs/remove-all-items-from-cart)
- [Remove all personal data from cart](https://developers.vtex.com/vtex-rest-api/docs/remove-all-personal-data)
- [Update cart items](https://developers.vtex.com/vtex-rest-api/docs/update-cart-items)
- [Add cart items](https://developers.vtex.com/vtex-rest-api/docs/add-cart-items)
- [Change the price of an SKU in a cart](https://developers.vtex.com/vtex-rest-api/docs/change-price)
- [Ignore profile data from cart](https://developers.vtex.com/vtex-rest-api/docs/ignore-profile-data)
- [Get cart installments information](https://developers.vtex.com/vtex-rest-api/docs/get-cart-installments-information)
- [Add coupons to the cart](https://developers.vtex.com/vtex-rest-api/docs/add-coupons-to-the-cart)
